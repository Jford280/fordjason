About Me

This entire site was built as an experiment. I wanted to create an Apache Server to host my own pages, and I wanted to learn more CSS. I built this server out of an old Atom based tablet, installing Ubuntu LTS - Hence the name ATOMSERVER. I installed the desktop version as a (just in case) but I do most updates via SSH. I was also able to get RDP and VNC installed, so that helps when I can't figure out how to get something done via commandline. I hope to post regular updates to these pages as I figure out more and more CSS.
